## 📋 Custom Post Type Structure pentru WordPress

### **GraphQL Schema:**

```graphql
type Service {
  id: ID!
  title: String!
  slug: String!
  content: String
  excerpt: String
  featuredImage: MediaItem
  
  # ACF Fields
  serviceDetails: ServiceDetails
  
  # SEO
  seo: SEO
}

type ServiceDetails {
  serviceIcon: String
  serviceGradient: String
  serviceFeatures: [ServiceFeature]
  serviceBenefits: String
  serviceFaq: [ServiceFaq]
  servicePrice: String
  serviceDeliveryTime: String
}

type ServiceFeature {
  featureTitle: String
  featureDescription: String
  featureIcon: String
}

type ServiceFaq {
  question: String
  answer: String
}
```
