## 📝 Note Implementare

### **Tehnologii Folosite în WordPress:**
- **Page Builder:** WPBakery (Visual Composer)
- **Slider:** Slider Revolution
- **Forms:** Contact Form 7
- **SEO:** Yoast SEO
- **Cache:** W3 Total Cache

### **Pentru Migrare la Next.js:**
1. ✅ Extrage conținut din shortcodes
2. ✅ Convertește la componente React
3. ✅ Recreează slider-ul cu Framer Motion
4. ✅ Migrează formulare la React Hook Form
5. ✅ Implementează SEO cu next-seo
6. ✅ Folosește GraphQL pentru conținut dinamic
